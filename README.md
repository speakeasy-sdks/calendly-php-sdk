## SDK Installation

### Composer

To install the SDK first add the below to your `composer.json` file:

```json
{
    "repositories": [
        {
            "type": "github",
            "url": "https://github.com/speakeasy-sdks/calendly-php-sdk.git"
        }
    ],
    "require": {
        "calendly/calendly": "*"
    }
}
```

Then run the following command:

```bash
composer update
```
